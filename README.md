##  Overview
A beginner‑friendly Python project for **string manipulation and text analysis**.  
It demonstrates operations like counting characters, slicing, replacing words, and finding substrings.

##  Features
- Count total characters in a string  
- Perform string slicing and indexing  
- Replace words or characters  
- Find substrings within text  
- Simple and clean Python code
